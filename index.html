IPA Evangelist

My name is Kin Lane. I am an API Evangelist. If you need more information on what an API is, header over to API Evangelist for the business of APIs, API Voice for the politics of APIs and to actually see some APIs, head over to API Stack.

But there is a darker, parallel universe to this fast growing API world. The world of India Pale Ales also known as IPAs.

A growing portion of individuals respondible for APIs, actually drink IPAs.

Mobile applications like Untappd that provide access to information about a variety of IPAs, also has an API.

Gatherings are popping up around the country, where people gather in pubs, drink IPAs and talk about APIs.

This site is dedicated to this phenomemon. The goal is track on on individuals, specific IPAs, breweries and pubs in this space.
